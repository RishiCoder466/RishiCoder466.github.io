Unit 2 Concepts

Concepts Covered

Variables

Variables are named storage locations in memory, used to hold data that can be changed during program execution. They allow for dynamic interaction within code.

Types

Types define the nature of data that a variable can hold. Common types include integers, strings, and booleans, each with distinct properties and behaviors.

Initialization

Initialization is the process of assigning an initial value to a variable. This ensures that variables start with a known state before being used in calculations.

Parameters

Parameters are special variables used in functions to receive input. They enable functions to operate on different data and customize their behavior based on provided values.

World Building

World Building in programming refers to creating a simulation or game environment where various elements interact. This involves designing the layout, characters, and rules of the world.

Arrays

Arrays are collections of elements, typically of the same type, stored in a single variable. They allow for efficient data management and access using indices, facilitating handling multiple values easily.
